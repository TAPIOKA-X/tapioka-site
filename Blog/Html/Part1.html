Github pagesのセットアップ


1. GitHub pagesの使い方

まず、「GitHub」にアクセスしてください。
そして、Sign up(登録)を選択して、メールアドレスを設定してください。登録したメールアドレス宛に確認メールが届くので、確認メールに記載されている、確認コードをGitHubに入力し登録を完了させてください。
次に、新しいリポジトリを作りましょう！
作り方は、右上にあるプラスマークをクリックして、New repositoryを選択して新しいリポジトリを作成してください。
新しいリポジトリに、index.htmlとabout.htmlを追加しましょう。
